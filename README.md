# BlackIronApp
微信小程序端

> 通过这个项目，你可以学习到如何做一个小程序，如何编写页面，请求后端数据，处理微信授权登录，小程序中上传文件，微信支付等等功能。

### 个人博客：
http://www.zjzjgxw.cn

### 相关项目：
* Spring boot 服务端程序地址 ：https://github.com/zjzjgxw/BlackIron

* Antd Pro 管理后台：https://github.com/zjzjgxw/BlackIron_Admin


### 功能点：
1. 微信授权登录
2. 商品展示
3. 微信支付
4. 评论（支持图片）
5. 商品展示支持视频
6. 分享小程序
